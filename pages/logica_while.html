Estrutura While

O que é o While?

Em tradução livre, While significa "Enquanto". Ele cria um laço que repete um bloco de código repetidamente enquanto uma condição for verdadeira.

Pense nele como um guarda de segurança que só deixa o código passar se a senha (condição) estiver correta.

O Fluxo de Decisão

Antes de cada repetição, o computador faz uma checagem:

A condição é Verdadeira?
Sim: Executa o código e volta ao início
Não: Encerra o laço e segue o programa

Atenção: Se a condição for falsa na primeira vez, o código nunca será executado.

Sintaxe e Exemplo

// Estrutura
while(condicao) {
  // Repete isto
}
// Contador real
let c = 0;
while(c < 3) {
  console.log(c);
  c++; // Importante!
}

Perigo: Loop Infinito

Se você esquecer de atualizar a variável (como o c++), a condição será sempre verdadeira.

O navegador tentará processar o código para sempre, o que causará o travamento da aba ou do sistema.
