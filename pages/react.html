Dominando React JS

O que é o React?

O React é uma biblioteca JavaScript para construir interfaces de usuário. Diferente do JS comum, ele foca na criação de componentes reutilizáveis.

Componentização

Imagine que cada parte do seu site (botão, menu, card) é um arquivo independente. No React, chamamos isso de componentes.

Vantagem: Você escreve o código uma vez e o reutiliza em todo o projeto, facilitando muito a manutenção.

O que é JSX?

É uma forma de escrever HTML diretamente dentro do JavaScript. Isso torna a criação de interfaces muito mais intuitiva.

function App() {
  return <h1>Bem-vindo ao React!</h1>;
}

Single Page Application (SPA)

O React cria SPAs. Isso significa que, ao navegar, o site não recarrega a página inteira, apenas troca o conteúdo central.

Isso proporciona uma experiência muito mais rápida e fluida para o usuário, parecida com um aplicativo de celular.
