Ambiente de Desenvolvimento

O que é uma IDE?

Sigla para Integrated Development Environment (Ambiente de Desenvolvimento Integrado).

É como uma "oficina completa" para o programador, reunindo tudo o que ele precisa para escrever, testar e rodar o código em um só lugar.

O Tripé da IDE

Para ser considerada uma IDE completa, o software geralmente possui:

Editor: Onde o texto é escrito (com cores!).

Compilador: Transforma o código em algo que o PC entende.

Debugger: Ferramenta para caçar erros no código.

IDE vs Editor de Código

Editor (ex: Notepad++): É como um bloco de notas com cores. Leve e rápido, mas você precisa fazer o resto "na mão".

IDE (ex: Visual Studio/Eclipse): É como uma fábrica automatizada. Mais pesada, porém faz quase tudo por você.

O VS Code está no meio termo: é um editor que "vira" IDE com extensões.

A Estrela: Visual Studio Code

O queridinho do mercado por ser leve e extremamente customizável.

Download em: code.visualstudio.com

O Poder das Extensões

O VS Code brilha porque você pode adicionar "superpoderes" a ele:

  • Portuguese (Brazil): Deixa a interface em português.
  • Themes: Muda as cores para não cansar a vista.
  • Live Server: Abre o site e atualiza sozinho ao salvar.
