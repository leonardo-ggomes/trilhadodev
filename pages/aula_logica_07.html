JavaScript Function

O que é uma Função?

Uma Function é um bloco de código projetado para executar uma tarefa específica. Pense nela como uma "receita" que você escreve uma vez e pode usar sempre que precisar.

  • Reutilização: Escreva o código uma vez, use várias.
  • Organização: Divide problemas grandes em partes menores.
  • Invocação: Ela só executa quando você a "chama".

Anatomia da Função

Para definir uma função, usamos a palavra-chave e um nome seguido de parênteses:

function nomeDaFuncao(parametro1, parametro2) {
  // A lógica fica aqui dentro
  return resultado;
}

Os parâmetros são como variáveis que a função recebe para trabalhar.

A palavra Return

O return para a execução da função e envia um valor de volta para quem a chamou.

function somar(a, b) {
  return a + b;
}

let resultado = somar(10, 5); // resultado será 15

Invocando (Chamando)

O operador ( ) é o que faz a mágica acontecer. Sem ele, você está apenas referenciando a função, não executando-a.

// Apenas referência:
let x = minhaFuncao;

// Execução real:
let y = minhaFuncao();
