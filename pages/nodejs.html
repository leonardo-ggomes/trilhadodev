Node.js: O Início

O que é o Node?

O Node.js® é um ambiente que permite executar JavaScript fora do navegador. Ele é o motor por trás de grandes sistemas como Netflix e LinkedIn.

  • Ultra veloz
  • Single Thread

Instalação

Para começar, instale a versão LTS (Long Term Support). Após instalar, abra seu terminal e digite:

C:\Users> node -v
Ir para Site Oficial

Seu primeiro código

Crie um arquivo app.js e escreva:

console.log("Servidor iniciado!");

Execute no terminal com: node app.js
