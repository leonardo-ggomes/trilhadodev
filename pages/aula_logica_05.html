Estruturas Condicionais

O que é If/Else?

É uma estrutura que permite ao seu código tomar decisões. Ele executa um bloco de comandos se uma condição for verdadeira, e outro caso contrário.

Pense nisso como uma bifurcação na estrada: você só segue um caminho por vez.

Sintaxe Básica

A condição fica entre parênteses e o código a ser executado entre chaves:

if (condicao) {
  // código se for verdadeiro
} else {
  // código se for falso
}

Exemplo de Uso

Vamos verificar se um herói é o Hulk:

var heroi = "Hulk";

if (heroi === "Thor") {
  console.log("Dominante");
} else if (heroi === "Hulk") {
  console.log("Introvertido!");
} else {
  console.log("Fraco");
}

Refinando a Condição

Você pode combinar múltiplas verificações usando operadores lógicos:

  • && (E): Todas as condições devem ser verdadeiras.
  • || (OU): Pelo menos uma deve ser verdadeira.
  • ! (NÃO): Inverte o valor booleano.
