JavaScript Arrays

O Problema da Escala

Imagine que você precisa guardar o nome de 300 filmes. Criar 300 variáveis (filme1, filme2...) seria um pesadelo, certo?

// Jeito errado e cansativo:
let filme1 = "Batman";
let filme2 = "Barbie";
let filme3 = "Top Gun";

O Array resolve isso! Ele é uma lista única que guarda tudo junto.

Como funciona o Índice?

No JavaScript, os Arrays são **indexados**. Isso significa que cada item tem um endereço numérico, e o mais importante: A contagem começa no ZERO.

Para acessar o "Batman", você chama o índice [0]. Para o "Barbie", o índice [1], e assim por diante.

Declarando seu primeiro Array

Usamos colchetes [ ] e separamos os itens por vírgulas.

const filmes = ["Batman", "Barbie", "Top Gun"];

// Lendo um valor:
console.log(filmes[1]); // Saída: "Barbie"

// Alterando um valor:
filmes[0] = "Spider-Man";

Arrays + Loops = ❤️

A verdadeira magia acontece quando usamos um loop para percorrer a lista inteira automaticamente.

for(let i = 0; i < filmes.length; i++) {
  console.log(filmes[i]);
}

Dica: .length retorna o tamanho total do array.
