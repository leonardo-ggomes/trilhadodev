Dominando Window Location

Anatomia da URL

URL é o endereço que identifica um recurso na rede. No JavaScript, o objeto window.location contém todas as informações do endereço atual.

Navegação via Script

Podemos forçar o navegador a mudar de página sem que o usuário clique em um link, útil após um login bem-sucedido, por exemplo:

window.location.href = "painel.html";

Isso substitui o endereço atual e redireciona o usuário instantaneamente.

Passando Dados via URL

O símbolo ? inicia a Query String. É assim que passamos informações de uma página para outra.

Exemplo: produtos.html?id=123

Aqui, estamos dizendo à página de produtos que queremos visualizar especificamente o item 123.

Capturando o Parâmetro

Para ler o que veio depois do ?, usamos o location.search.

var query = location.search;
var cleanId = query.replace('?', '');

O método replace é usado para remover o ponto de interrogação e deixar apenas o valor puro para sua lógica.

Por que usar isso?

Imagine um site de notícias. Você não cria 1.000 páginas HTML. Você cria uma página noticia.html que carrega o texto baseado no ID que vem pela URL!
