Estrutura For

O Loop Organizado

Diferente do while, onde as peças ficam espalhadas, o For concentra a inicialização, a condição e o incremento em um único lugar.

Use o For sempre que você souber o número total de repetições antecipadamente.

Anatomia do Comando

O for se divide em três partes essenciais separadas por ponto e vírgula:

for ( let i = 0; i < 10; i++ ) { ... }
  • Início: Onde o contador começa (roda 1 vez).
  • Condição: Enquanto for verdade, o loop continua.
  • Passo: Como o contador muda a cada volta.

Na Prática

Veja como imprimir números de 1 a 5 no console de forma elegante:

// Começa em 1, vai até 5, aumenta de 1 em 1
for (let i = 1; i <= 5; i++) {
  console.log("Número: " + i);
}

Lembre-se: as expressões são opcionais, mas os dois pontos e vírgulas ;; são obrigatórios!
