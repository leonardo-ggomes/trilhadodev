Banco de Dados Relacional

O que é?

Um banco de dados relacional organiza informações em tabelas que se conectam entre si. Ele é o "cérebro" onde a aplicação guarda tudo o que não pode ser esquecido.

O que é um SGBD?

O Sistema Gerenciador de Banco de Dados é o software que controla o acesso aos dados.

Exemplos Famosos: MySQL, PostgreSQL, Oracle e SQL Server.

Imagine o banco como um arquivo físico e o SGBD como o bibliotecário que organiza e busca os livros para você.

Estrutura de Dados

No modelo relacional, usamos:

  • Colunas: Definem o tipo de dado (Nome, Idade, Email).
  • Linhas: São os registros reais (ex: o usuário "João").
  • Primary Key (PK): Um ID único que nunca se repete.

A Linguagem SQL

Structured Query Language é como conversamos com o banco.

SELECT * FROM usuarios WHERE id = 1;

Com esse comando, pedimos ao banco: "Me mostre todos os dados do usuário com ID número 1".

Por que Relacional?

Ele garante a integridade. Você não consegue deletar um "Pedido" se ele estiver ligado a um "Cliente" que ainda existe.
