Introdução ao DOM API

O que é o DOM?

DOM significa Document Object Model. É a ponte que permite ao JavaScript "conversar" com o seu HTML.

Através dele, podemos alterar textos, cores e até criar novos elementos dinamicamente.

A Árvore de Elementos

O navegador transforma seu HTML em uma árvore de objetos. Cada tag vira um nó.

O objeto raiz de tudo no seu site é o document.

Como selecionar?

Para o JS agir, ele precisa localizar o elemento. O método mais eficaz é usar o ID:

// HTML: <h1 id="titulo">Olá</h1>

const h1 = document.getElementById("titulo");

A variável h1 agora dá controle total sobre aquela tag específica.

Alterando a Página

Depois de selecionar, você pode mudar o conteúdo ou o estilo visual via código:

h1.innerHTML = "Bem-vindo ao DOM!";
h1.style.color = "blue";
h1.classList.add("destaque");

Eventos do Usuário

O DOM permite reagir a cliques, movimentos de mouse e teclas digitadas usando o addEventListener.
