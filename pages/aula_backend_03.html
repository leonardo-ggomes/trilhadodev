Servidor: Conceito

O que é um servidor?

Um servidor é um software que "escuta" requisições da internet e envia respostas. No Node.js, o módulo http é quem faz esse trabalho pesado.

A estrutura básica

Com apenas estas linhas, você coloca um servidor de pé:

const http = require('http');

http.createServer((req, res) => {
  res.write('Ola Mundo!');
  res.end();
}).listen(8080);

Portas e Localhost

O comando .listen(8080) reserva um canal no seu computador para o servidor.

Acesse: Abra seu navegador e digite http://localhost:8080 para ver o resultado.
