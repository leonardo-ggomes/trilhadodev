CSS Flexbox

O que é Flexbox?

É um sistema de layout unidimensional para CSS que permite organizar elementos em linhas ou colunas de forma automática e inteligente.

A mágica acontece quando você usa display: flex; no elemento pai!

O Pai e os Filhos

O Flexbox trabalha com dois níveis hierárquicos:

Flex Container

O elemento pai. Ele define as regras de alinhamento para seus filhos.

Flex Items

Os filhos diretos. Eles se ajustam conforme as regras do pai.

Entendendo os Eixos

O segredo do Flexbox está em entender os dois eixos invisíveis:

  • Main Axis (Eixo Principal): Onde os itens são distribuídos (Horizontal por padrão).
  • Cross Axis (Eixo Transversal): O eixo perpendicular ao principal (Vertical por padrão).

Propriedades Essenciais

justify-content: center; /* Alinha no eixo principal */
align-items: center;     /* Alinha no eixo transversal */
flex-direction: column;  /* Muda o eixo para vertical */
gap: 20px;               /* Espaço entre os itens */
