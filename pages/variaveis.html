JavaScript Variáveis

O que é uma Variável?

Uma variável é um espaço na memória do computador destinado a guardar informações. Pense nela como uma caixa etiquetada onde você guarda um valor para usar depois.

  • Identificação: Cada caixa tem um nome único.
  • Atribuição: Usamos o sinal = para colocar algo na caixa.
  • Mutabilidade: O conteúdo pode mudar (exceto em constantes).

Formas de Declarar

No JavaScript moderno, focamos principalmente em let e const:

let

Para valores que podem mudar durante o programa. Tem escopo local.

const

Para valores fixos. Uma vez definida, você não pode alterar o valor.

let preco = 29.90; // Pode mudar
const taxaFixa = 0.05; // Erro se tentar mudar

Tipos de Dados Primitivos

As variáveis podem guardar diferentes "espécies" de informação:

Tipo Exemplo Descrição
String "Olá" Textos (sempre entre aspas).
Number 10 ou 5.5 Números inteiros ou decimais.
Boolean true / false Valores lógicos (Sim/Não).

Regras de Ouro 🏆

  • Camel Case: Use nomeCompleto em vez de nome_completo.
  • Sem espaços: Nunca use espaços ou acentos nos nomes.
  • Sem números no início: 1item é proibido; use item1.
  • Semântica: O nome deve dizer o que a variável faz. Evite let x = 10;.
