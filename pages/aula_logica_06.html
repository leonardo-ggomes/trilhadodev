Estrutura Switch

O Escolhedor de Caminhos

O Switch é uma instrução condicional que executa diferentes ações baseadas em múltiplos casos possíveis. É uma alternativa elegante ao if...else if quando temos muitas opções.

  • Compara um valor com vários casos.
  • Executa o bloco se houver correspondência.
  • Usa um caminho padrão (default) se nada coincidir.

Anatomia do Switch

Veja como organizar os blocos de verificação:

switch(expressao) {
  case x:
    // código aqui
    break;
  case y:
    // código aqui
    break;
  default:
    // se nenhum bater
}

Exemplo Real: Escolha de Filme

let filme = "batman";

switch(filme) {
  case "spiderman":
    console.log("Vai teia!");
    break;
  case "batman":
    console.log("Eu sou a vingança!");
    break;
  default:
    console.log("Filme não encontrado.");
}

A Importância do Break

A palavra-chave break é vital. Sem ela, o JavaScript continua executando os casos abaixo mesmo que eles não correspondam ao valor!

O default não precisa de break, pois ele é sempre a última opção do bloco.

Desafio Switch
