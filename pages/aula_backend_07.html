Entendendo o JSON

O que é JSON?

JavaScript Object Notation é um formato leve para troca de dados.

Embora tenha "JavaScript" no nome, ele é independente de linguagem e usado por quase todas as tecnologias modernas para enviar informações entre servidor e navegador.

Por que usar JSON?

  • Leve: Arquivos pequenos que carregam rápido.
  • Legível: Fácil para humanos lerem e computadores processarem.
  • Universal: Funciona em Python, Java, PHP, Node.js, etc.

Chave e Valor

A estrutura básica do JSON é composta por pares de "chave": "valor" dentro de chaves { }.

{
  "nome": "Leonardo",
  "idade": 25
}

As chaves devem estar sempre entre aspas duplas!

Tipos de Dados no JSON

JSON aceita strings, números, booleanos, arrays e até outros objetos:

{
  "titulo": "O Pequeno Príncipe",
  "ano": 1943,
  "eFiccao": true,
  "generos": ["Novela", "Infantil"]
}
