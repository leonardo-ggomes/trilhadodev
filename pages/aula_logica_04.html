Git & GitHub

Sobre o GitHub

O GitHub é uma plataforma na nuvem que hospeda seus projetos. É como o "Google Drive" dos programadores, mas focado em código e colaboração social.

Ele permite que pessoas do mundo todo contribuam com o seu software.

Conceitos Git

O Git é a ferramenta que você instala no PC. Ele funciona como uma "Máquina do Tempo" para o seu código.

  • Versionamento: Salva fotos (commits) do seu projeto em diferentes estados.
  • Segurança: Se você errar algo hoje, pode voltar exatamente para o que funcionava ontem.

Configuração Inicial

No seu terminal, você deve definir quem é o autor das alterações:

// Identifique-se para o sistema
$ git config --global user.name "Seu Nome"
$ git config --global user.email "seu@email.com"

Essa configuração só precisa ser feita uma vez por computador.

O Ciclo Local

Sempre que terminar uma tarefa, você segue este fluxo no terminal:

# 1. Prepara os arquivos modificados
$ git add .

# 2. Salva a versão com uma mensagem explicativa
$ git commit -m "ajuste no estilo do cabeçalho"

Fluxo Remoto

Para enviar suas alterações locais para o GitHub (nuvem), usamos o comando Push:

# Envia para o servidor remoto
$ git push origin main

E para baixar as novidades de outros desenvolvedores:

$ git pull origin main
