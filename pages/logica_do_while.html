Estrutura Do-While

Atire primeiro, pergunte depois!

Diferente do while comum, o Do-While garante que o bloco de código seja executado pelo menos uma vez antes de testar a condição.

Útil quando você precisa que uma ação ocorra (como exibir um menu) antes de verificar se ela deve se repetir.

While vs Do-While

Entenda a ordem de execução:

While: Teste → Execução
Do-While: Execução → Teste

Como escrever

do {
  // código executado primeiro
  i++;
} while (i < 5);

Note o ponto e vírgula ; após o while — ele é obrigatório nesta estrutura!
