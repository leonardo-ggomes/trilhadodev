Entendendo APIs

O que é uma API?

API significa Application Programming Interface.

É uma ponte que permite que dois softwares conversem de maneira organizada e padronizada.

A Analogia do Garçom

No restaurante do desenvolvimento:

  • Você: O cliente (Front-end).
  • Garçom: A API (Leva o pedido).
  • Cozinha: O Servidor (Processa os dados).

Request & Response

O ciclo de vida de uma API:

1. Request: O app pede algo (ex: lista de produtos).

2. Response: O servidor responde (ex: JSON com os preços).

API com Node.js


const http = require('http');
const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'application/json' });
  res.end(JSON.stringify({ status: "Rodando" }));
});
server.listen(3000);
